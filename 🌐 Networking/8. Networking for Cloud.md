### 3пёҸвғЈ **Ports**

- Multiple apps on one server need **different entry points**
- Ports range: `1 вҖ“ 65535`
- Common ports:
    - `80` вҶ’ HTTP
    - `443` вҶ’ HTTPS

### 4пёҸвғЈ **Network Segmentation (Subnets)**

- Split network into **isolated sections** for security
- Examples:
    - Public subnet вҶ’ Frontend
    - Private subnet вҶ’ App servers

- Reduces blast radius in case of attack рҹ”җ

### 5пёҸвғЈ **Routing**

- Controls **how traffic moves between subnets**

- Required for communication across segments

---

### 6пёҸвғЈ **Firewalls**

- Control **what traffic is allowed**
- Types:
    - **Host firewall** вҶ’ Protects a single server
    - **Network firewall** вҶ’ Filters traffic between subnets
- Example rule:
    - DB accepts traffic **only on port 3306** from app subnet
- Security is always **layered** рҹӣЎпёҸ

### 7пёҸвғЈ **Private vs Public IP**

- **Private IPs**:
    - Used inside network
    - Not reachable from internet
- **Public IPs**:
    - Reachable from anywhere
    - Limited & costly

---

### 8пёҸвғЈ **NAT (Network Address Translation)**

- Allows **private servers to access the internet**
- Many private IPs вҶ’ **1 public IP**
