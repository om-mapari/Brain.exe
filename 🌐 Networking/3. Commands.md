## 1️⃣ DNS Lookup

```bash
nslookup om-mapari.com
```

- Resolves domain → IP
- Uses DNS


![[../images/1-Networking3. Commands.webp]]

---

## 2️⃣ Connectivity Check

```bash
ping om-mapari.com
```

- Checks reachability + latency
- Uses ICMP


![[../images/2-Networking3. Commands.webp]]

---

## 3️⃣ Route Path

```bash
tracert om-mapari.com
```

- Shows hops (routers)
- Network Layer (L3)

---

## 4️⃣ Public IP

```bash
curl ipinfo.io
```

- Shows your public IP


![[../images/3-Networking3. Commands.webp]]

---

## 5️⃣ HTTP Check

```bash
curl -I https://om-mapari.com
```

- Checks response headers


![[../images/4-Networking3. Commands.webp]]
