## 3пёҸвғЈ **NAT Gateway (Network Address Translation)**

вң” **Definition:**  
A **NAT Gateway** allows **instances in private subnets** to access the **Internet** (for updates, APIs, etc.)  
вһЎпёҸ while **preventing inbound traffic** from the Internet.

вң” **How It Works:**

1. Private instance вҶ’ sends request to Internet
2. NAT Gateway вҶ’ replaces private IP with **its public IP**
3. Response вҶ’ comes back to NAT Gateway вҶ’ forwarded to private instance


вң” **Key Points:**

- Deployed in a **public subnet**
- Assigned a **public Elastic IP**
- One-way communication: **Outbound only**
- Used for **security + outbound internet access**


рҹ§  **Example Use Case:**  
Private EC2 вҶ’ needs to `yum update` вҶ’ request goes via NAT Gateway вҶ’ to Internet.
