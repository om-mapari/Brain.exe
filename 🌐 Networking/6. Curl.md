## Use Guide

| Goal            | Command                                 |
| --------------- | --------------------------------------- |
| Debug request   | `curl -v`                               |
| Check status    | `curl -I`                               |
| API call 🔥     | `curl -X + -d + -H`                     |
| Follow redirect | `curl -L`                               |
| Save file       | `curl -o file.html https://example.com` |
| Ignore SSL ⚠️   | `curl -k`                               |

---

## 1️⃣ Basic Request

```bash
curl https://example.com
```

- Fetch full response body


![[1-6. Curl.webp]]

---

## 2️⃣ Headers Only 📌

```bash
curl -I https://example.com
```

- Check status code + headers


![[2-6. Curl.webp]]

---

## 3️⃣ Follow Redirects

```bash
curl -L https://example.com
```

- Handles 301/302 redirects

---

## 4️⃣ Custom HTTP Method

```bash
curl -X POST https://example.com/api
```

- Supports `GET`, `POST`, `PUT`, `DELETE`

---

## 5️⃣ Send Form Data

```bash
curl -X POST https://example.com/api \
  -d "name=omi&age=25"
```

---

## 6️⃣ JSON API Call 

```bash
curl -X POST https://example.com/api \
  -H "Content-Type: application/json" \
  -d '{"name":"omi"}'
```

---

## 7️⃣ Add Headers

```bash
curl -H "Authorization: Bearer token" \
     -H "Accept: application/json" \
     https://example.com
```

---

## 8️⃣ Ignore SSL

```bash
curl -k https://example.com
```

- Skip SSL verification (debug only)

---

## 🔥 Most Useful Combo

```bash
curl -v -L https://example.com
```
