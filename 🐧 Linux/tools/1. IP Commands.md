# IP Commands

## DNS Lookup with nslookup

```bash
mapar@CoderOM MINGW64 ~
$ nslookup 13.233.199.251
Server: dns.google
Address: 2001:4860:4860::8888

Name: ec2-13-233-199-251.ap-south-1.compute.amazonaws.com
Address: 13.233.199.251
```

> [!info]
> The `nslookup` command performs reverse DNS lookup to find the hostname associated with an IP address.
