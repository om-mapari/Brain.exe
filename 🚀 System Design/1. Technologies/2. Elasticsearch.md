#### How can you use ES in system desing interview

- Typically used for complex search scenario
- It's not your primary database. CDC push the updates from db into ES  
- work best with read-heavy workloads. not good if write is exceeding reads
- We will have eventual consistency. It will be little behind primery db.
- do we really need it? -- if really then only use it.
- don't use if complex joins

Often use DB + sync into Elasticsearch.
