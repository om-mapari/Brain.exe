# Fargate vs EC2 Launch Types

## EC2 Launch Type

In EC2 launch type:
- you own the underline ec2 instances
- you manage them
- you configure them
- you install docker ecs agent firewall patches

> [!info]
> You get full control of the infrastructure. Only thing ECS does is it manages the container with ecs agent.

## ECS Fargate
