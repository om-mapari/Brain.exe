## Pydentic

- like zed for python or typescript of python 
- used for data validation
- used for setting management
- data parsing 
- data serilization deserilization 
- api development 

```python
from pydantic import BaseModel

class User(BaseModel):
    id: int
    name: str

    is_active: bool



input_data = {

    'id' : 101,

    'name' : "Chaicode",

    'is_active': True

}



data = User(**input_data)
```
