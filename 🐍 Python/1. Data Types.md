## 1️⃣ Core Differences

| Feature              | List                             | Tuple                        | Set                           |
| -------------------- | -------------------------------- | ---------------------------- | ----------------------------- |
| Ordered              | ✅                                | ✅                            | ❌                             |
| Allows Duplicates    | ✅                                | ✅                            | ❌                             |
| Mutable (modifiable) | ✅                                | ❌                            | ✅                             |
| Index Access         | ✅                                | ✅                            | ❌                             |
| Syntax               | `[1, 2, 3]`                      | `(1, 2, 3)`                  | `{1, 2, 3}`                   |

---

## 2️⃣ When to Use What

- **List**
    - Need order + duplicates
    - Need to modify data
    - Most commonly used
- **Tuple**
    - Data should not change
    - Safer and faster than lists
    - Example: fixed values like dates, coordinates
- **Set**
    - Need only unique values
    - Fast duplicate removal
    - Order does not matter

---

## 3️⃣ Key Examples

### Remove duplicates using set

```python
nums = [1, 2, 2, 3]
unique = list(set(nums))
```

### Lists are mutable

```python
l = [1, 2]
l.append(3)
```

### Tuples are immutable

```python
t = (1, 2)
t[0] = 5  # ❌ error
```

### Sets have no index no duplicates

```python
s = {1, 2, 3, 3}
print(s) # {1, 2, 3}
s[0]  # ❌ error
```
