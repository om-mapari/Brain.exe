# Excalidraw Shortcuts Cheat Sheet

## 1️⃣ Basic Tools

- `V` → Selection tool
- `R` → Rectangle
- `4` → Ellipse (circle)
- `D` → Diamond
- `A` → Arrow
- `L`/`6` → Line
- `X` → Pencil (free draw) ✏️
- `double click` → Text
- `T` → Text

---

## 2️⃣ Navigation & View

- `Space + Drag` → Pan canvas
- `Shift + 1` → Zoom to fit

---

## 3️⃣ Tricks

- `Alt + Drag` → Duplicate ⭐
- `Shift + Click` → Multi-select
- `Esc` → Exit current tool
-  Combine **`Shift` + shapes** → clean diagrams
- `Shift + Drag` → Maintain proportion for resizing while multiple selected object 
- `Multi-point Arrow` →
    - Select **Arrow tool**
    - Then do:
        - **Click → release**
        - Move cursor
        - **Click → release**
        - Hold Shift to maintain the angle
        - Move cursor
        - **Click → release**
    - Press `Esc` to finish
- `Single Point -> multi point` → hover to mid point of arrow then Ctrl + click 
- `ctrl + alt + c` →  copy style  `ctrl + alt + v` → paste style

---

## 4️⃣ Alignment & Arrangement

- `Ctrl + G` → Group elements → Firest multi-select with `Shift + Click` then Group elements 
- `Ctrl + ]` → Bring forward
- `Ctrl + [` → Send backward
