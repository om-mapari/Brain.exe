# because I will forget this 🧠

This is a collection of notes I write for myself.

I forget things. A lot.  
So whenever I learn something, think about something, or struggle to understand something,
I write it down here.

This repository is basically a backup system for a very unreliable memory.

These notes will also be available as a website.

**Website:** https://brain.om-mapari.com/  

---

## Philosophy

- Notes > memory
- Write first, organize later
- If it’s not written down, it doesn’t exist
- Clarity comes from writing, not thinking
- Forgetting is normal, notes are insurance
- Capture now, refine later
- Messy notes are better than no notes
- Documentation is thinking out loud
